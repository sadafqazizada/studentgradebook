# Student Gradebook Manager

## Project Description

Student Gradebook Manager is a Python Object-Oriented Programming (OOP) project for managing students, courses, assessments, and grades.

The system allows users to:

- Add new students
- View student information
- Add new courses
- Enroll students in courses
- Add assessments (Quiz, Exam, Project)
- Record student grades
- Calculate average grades
- Display student reports
- Search students
- Update student information
- Delete students

## OOP Concepts Used

- Classes and Objects
- Encapsulation
- Inheritance
- Method Overriding
- Constructors
- Getters and Setters

## Creative Features

- Letter Grade (A, B, C, D, F)
- Student Ranking based on average grades
